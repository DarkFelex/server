API сервера: Работа с сессией "/api/v1/sessions" Метод GET - получить данные о залогиненном пользователе Код отвера: 200 - если аторизован Тело: json с данными о текуцем пользователе Код отвера: 401 - если не авторизован (нет совпадения sessionId на login) Метод POST - залогиниться параметры: login = логин пользовател pass = пароль пример: localhost:8080/api/v1/sessions?login='test'&pass='test' Код ответа: 200 - если удалось залогиниться (пользователь с текущим sessionId считается залогиненным) Тело: json с данными о текуцем пользователе Код ответа: 401 - если логин или пароль неверные Код ответа: 400 - не указан или пустой логин или пароль Метод DELETE - разлогиниться Код ответа: 200 - разлогиниться удалось успешно Код ответа: 401 - если пользователь не был залогинен Зарегистрироваться "/signup" Метод POST параметры: login = логин пользовател pass = пароль пример: localhost:8080/signup?login=user1&pass=123456 Код ответа: 200 - пользователь зарегистрирован Код ответа: 400 - не указан или пустой логин или пароль Код ответа: 409 - такой логин уже существует Залогиниться "/signin" Метод POST - залогиниться параметры: login = логин пользовател pass = пароль пример: localhost:8080/signin?login='test'&pass='test' Код ответа: 200 - если удалось залогиниться (пользователь с текущим sessionId считается залогиненным) Тело: "Authorized: login_name" Код ответа: 400 - если логин или пароль пуст или отсутствует в запросе Код ответа: 401 - логин или пароль указаны неверно Разлогиниться "/signout" Метод GET пример: localhost:8080/signout Код ответа: 200 - разлогин произошёл успешно Создать деревню "/api/v1/create_village" ещё не реализован Получить информацию о деревне "/api/v1/get_village" ещё не реализован
